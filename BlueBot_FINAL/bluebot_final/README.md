# 🤖 BlueBot — App Blocker
> FIRST Tech Challenge — Reims

## Télécharger l'application

1. Va dans l'onglet **Actions** sur GitHub
2. Clique sur le dernier build vert ✅
3. Télécharge **BlueBot-Windows** (PC) ou **BlueBot-Android** (téléphone)

### Windows
Double-clique sur `BlueBot.exe` — c'est tout.

### Android
Active l'installation depuis sources inconnues sur ton téléphone, puis ouvre le fichier `.apk`.
